CyberDeck — open-source ПО для удаленного управления собственным ПК в локальной сети.
Используя проект, вы подтверждаете согласие с условиями ниже.

## 1. Заявление о безопасности

### 1.1 Не является вредоносным ПО

Проект не позиционируется как вирус, троян, spyware или майнер.
Исходный код открыт для аудита и не должен содержать скрытые backdoor-механизмы.

### 1.2 Без скрытого наблюдения

- Нет «невидимого» режима работы.
- Нет целевой отправки пользовательских данных третьим сторонам.
- Рабочий трафик предназначен для локальной сети между вашими устройствами.

### 1.3 Открытый код

Код доступен публично для проверки логики, сетевого поведения и модели доступа.

## 2. Допустимое и недопустимое использование

### 2.1 Допустимо

- личное использование;
- управление собственными устройствами;
- обучение и автоматизация.

### 2.2 Запрещено

- использование без согласия владельца устройства;
- шпионаж и кража данных;
- нарушение работы чужих систем;
- иные противоправные действия.

Пользователь несет полную ответственность за законность применения ПО.

## 3. Конфиденциальность

- Приложение не предназначено для централизованного сбора личных данных.
- Для базовой работы достаточно локальной сети.
- Видеоданные обрабатываются как поток; долговременное хранение зависит от ваших настроек и ОС.

## 4. Отказ от гарантий и ограничение ответственности

ПО предоставляется «как есть», без прямых и подразумеваемых гарантий.
Разработчик не несет ответственности за прямой/косвенный ущерб, возникший из использования или невозможности использования проекта, в пределах, допустимых применимым правом.

## 5. Принятие условий

Запуск и использование CyberDeck означает принятие этих условий.
Если вы не согласны с ними, прекратите использование ПО.
